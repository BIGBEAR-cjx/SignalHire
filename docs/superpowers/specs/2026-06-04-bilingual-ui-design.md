# 中英双语切换设计

## 目标

为 SignalHire 增加中文 / English 双语版本，用户可在界面中自行切换。第一版重点覆盖产品主路径：公开首页、登录注册、工作台导航、总览、项目、搜人、核验台、候选池、历史和设置。

## 方案

采用轻量字典方案，不引入大型 i18n 路由框架。新增 `zh/en` 字典、全局 `LanguageProvider` 和语言切换控件。默认语言为中文，用户切换后写入 `localStorage`，刷新后保持。

## 范围

- UI 固定文案按当前语言展示。
- 运行中任务、历史记录、项目名称、候选人姓名、用户输入、后端返回的候选人资料保留原始内容。
- 搜索 / 核验请求会携带当前界面语言，后端 prompt 会要求新生成报告按该语言输出。
- 暂不做 `/en` 路由、数据库级用户偏好、自动浏览器语言检测和历史报告自动翻译。

## 验证

- `i18n.test.mjs` 覆盖语言归一化、支持语言判断、字典回退。
- `tsc --noEmit` 覆盖 TypeScript 接入。
- `npm run build` 覆盖 Next.js 构建。
